# Installscript voor macOS & Arch Linux

![Shell Script](https://img.shields.io/badge/Shell-Bash-4EAA25?logo=gnubash&logoColor=white)
![macOS](https://img.shields.io/badge/Platform-macOS-000000?logo=apple)

![Status](https://img.shields.io/badge/Status-Work_in_progress-yellow)

Een simpel maar krachtig installatiescript om **snel een nieuw macOS- of Arch Linux-systeem op te zetten**.  
Ideaal bij een herinstallatie of wanneer je meerdere systemen wilt beheren met dezelfde setup.

Het script detecteert automatisch welk OS je gebruikt en runt de juiste installatie- en configuratiestappen.

Clone deze repository en voer het hoofdscript uit:

```sh
sh install.sh
```
